# ACL submission paper

Fresh ACL-style manuscript for the decision-surface study:

> **Search APIs Are Decision Surfaces: A Controlled Study of Tool-Using QA Agents**

The old draft should be treated as superseded. This folder now keeps the paper, deterministic number macros, Graphviz DOT sources, generated TikZ vector figures, and a source-data audit script in one reproducible path.

## Build

```bash
cd paper
make        # regenerate render-only figures/macros, then build PDF
make quick  # one pdflatex pass
```

The build path uses `figures/make_figures.py --render-only`, which does not need the large raw JSONLs. It writes the last validated constants into `figures/numbers.tex` and regenerates all DOT-derived TikZ figure files.

## Full metric audit

The raw traces and judge outputs are stored through Git LFS. To recompute every metric from raw data:

```bash
git lfs pull
cd paper
make audit
```

`make audit` runs `python3 figures/make_figures.py` without `--render-only`. It consumes:

- `data/traces/phase1_v1_{brave,tavily,firecrawl}_gpt54_fetch_tool_jina_100.jsonl`
- `results/llm_judge/kimi_document_judge_surface_v3_{brave,tavily,firecrawl}_100_all_visible.jsonl`
- `results/provider_comparison/brave_tavily_firecrawl_fetch_tool_jina/provider_per_query.jsonl`
- `results/provider_comparison/brave_tavily_firecrawl_fetch_tool_jina/provider_summary.json`

If any raw JSONL is still a Git LFS pointer, the script exits with an explicit `git lfs pull` instruction rather than silently auditing incomplete data.

## Submission mode

`main.tex` uses `\usepackage[review]{acl}` for anonymous ACL review submission. Switch to `preprint` or `final` only outside anonymous review.

## Metric spine

The paper centers on four quantities, all computed by `figures/make_figures.py` in audit mode:

1. final answer quality (EM / F1),
2. visible support over judged snippet/page surfaces,
3. the SMART / MISSED / BLIND / NO-OP decision partition,
4. the answer-independent contradict-to-gold ratio over snippet-only URL rows.

The figures are intentionally rendered from Graphviz DOT into TikZ rather than hand-positioned raster images, so layout is deterministic and review-safe.
